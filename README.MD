# M-power 迷你电源项目

在宿舍调试设备没有可调电源，实验室的可调电源体积大并且算是违禁物品，所以就想自己做一个可调电源，要足够迷你，能够满足调试设备的需求，并且具备恒压、恒流功能、有便携性、支持PD协议以及常见DC电源接口，XT60接口（后面可能准备看一下无人机相关的东西）。
